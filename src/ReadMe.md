GitHub Crash Course

Always log your code at the end of the night/work session using the following
git add .
git commit -m"<your message about the work>"
git push origin <branch name>

to pull most recent changes from main
git pull origin main

to make a new branch
git checkout -b <branch name>
Typically this will be a shorthand descript on the branch purpose
i.e. git checkout -b fileReader
